
# Roadmap

Dette dokumentet skisserer mulige videreutviklinger av NorMacro.

## Kort sikt

- `plot_business_cycle()`
- bedre presentasjon av korrelasjonsmatriser
- flere eksempler i README
- mer dokumentasjon av analysefunksjoner

## Mellomlang sikt

- pakke NorMacro som en ordinær R-pakke
- vurdere `pkgdown`-side
- vurdere Shiny-dashboard
- legge til vignetter
- mer avanserte konjunktur- og trendmetoder

## Mulige metadatautvidelser

- `Display_navn`
- `Kildefrekvens`
- `Nøkkelord`
- `Sist_oppdatert`
- `Revisjonsnotat`

## Prinsipper videre

NorMacro bør fortsatt prioritere:

- representative indikatorer fremfor flest mulig variabler
- transparente beregninger
- enkel og stabil API
- full metadata
- testbare funksjoner
- standard R-objekter
